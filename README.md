# README Generator

## Description

A Node.js console app that prompts a user for information about their project, then generates a README file for them in the current repository.

## Table of Contents

  - [Installation](#Installation)

  - [Usage](#Usage)

  - [License](#License)

  - [Contributing](#Contributing)

  - [Tests](#Tests)

  - [Questions](#Questions)

## Installation

- Download Node.js
- Initialize Node Package Manager: npm init -y
- Install dependencies: npm i

## Usage

Install this folder into your application and run index.js to create a README from the console.
